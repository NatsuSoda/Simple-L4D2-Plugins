## 配置指令 (ConVar)

- `l4d_scs_auto_model` 默认值: `1` - 开关8人独立模型?
- `l4d_scs_tab_hud_bar` 默认值: `1` - 在哪些地图上显示一代人物的TAB状态栏? \n0=默认, 1=一代图, 2=二代图, 3=一代和二代图.
- `l4d_csm_admin_flags` 默认值: `z`
- `l4d_csm_in_transition` 默认值: `1` - 启用8人独立模型后不对正在过渡的玩家设置?
