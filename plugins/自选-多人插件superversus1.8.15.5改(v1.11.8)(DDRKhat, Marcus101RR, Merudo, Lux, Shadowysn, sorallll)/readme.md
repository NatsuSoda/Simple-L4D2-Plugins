- `sm_bot` [管理员指令] (权限: ADMFLAG_ROOT)

## 配置指令 (ConVar)

- `bots_limit` 默认值: `4` - 开局Bot的数量
- `bots_join_limit` 默认值: `-1`
- `bots_join_flags` 默认值: `3` - 额外玩家加入生还者的方法. \n0=插件不进行处理, 1=输入!join手动加入, 2=进服后插件自动加入, 3=手动+自动.
- `bots_join_respawn` 默认值: `1`
- `bots_spec_notify` 默认值: `3` - 完全旁观玩家点击鼠标左键时, 提示加入生还者的方式 \n0=不提示, 1=聊天栏, 2=屏幕中央, 3=弹出菜单.
- `bots_give_slot0` 默认值: `131071`
- `bots_give_slot1` 默认值: `1064`
- `bots_give_slot2` 默认值: `0`
- `bots_give_slot3` 默认值: `1`
- `bots_give_slot4` 默认值: `3`
- `bots_give_type` 默认值: `2`
- `bots_give_time` 默认值: `0` - 什么时候给玩家装备. \n0=每次出生时, 1=只在本插件创建Bot和复活玩家时.
