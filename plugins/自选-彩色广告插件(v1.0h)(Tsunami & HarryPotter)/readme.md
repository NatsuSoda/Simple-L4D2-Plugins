# 自选-彩色广告插件(v1.0h)(Tsunami & HarryPotter)

## 简介

Display advertisements

## 包含文件

- Advertisements
- 未知
- 未知

## 配置指令 (ConVar)

- `sm_advertisements_enabled` 默认值: `1` - 开启广告插件. 0=禁用, 1=启用.
- `sm_advertisements_file` 默认值: `advertisements.txt` - 设置广告文件名称.
- `sm_advertisements_interval` 默认值: `30` - 设置广告的播放间隔/秒.
- `sm_advertisements_soundfile` 默认值: `ui/beepclear.wav`
