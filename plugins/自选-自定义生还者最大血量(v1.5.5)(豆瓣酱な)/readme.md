# 自选-自定义生还者最大血量(v1.5.5)(豆瓣酱な)

## 简介

自定义生还者血量

## 配置指令 (ConVar)

- `l4d2_first_aid_heal_percent` 默认值: `0.8`
- `l4d2_adrenaline_health_buffer` 默认值: `50`
- `l4d2_first_aid_kit_max_heal` 默认值: `200`
- `l4d2_pain_pills_health_value` 默认值: `100`
- `l4d2_z_survivor_respawn_health` 默认值: `100`
- `l4d2_survivor_revive_health` 默认值: `60`
