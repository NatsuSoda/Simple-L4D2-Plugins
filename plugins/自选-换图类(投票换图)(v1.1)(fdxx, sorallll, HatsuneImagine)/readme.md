# 自选-换图类(投票换图)(v1.1)(fdxx, sorallll, HatsuneImagine)

## 简介

自选-换图类(投票换图)(v1.1)(fdxx, sorallll, HatsuneImagine)

## 可用指令

- `sm_v3`
- `sm_maps`
- `sm_chmap`
- `sm_vmap`
- `sm_votemap`
- `sm_mapvote`
- `sm_mapnext`
- `sm_votenext`
- `sm_reload_vpk` [管理员指令] (权限: ADMFLAG_ROOT)
- `sm_update_vpk` [管理员指令] (权限: ADMFLAG_ROOT)
- `sm_missions_reload` [管理员指令] (权限: ADMFLAG_ROOT)
- `sm_missions_export` [管理员指令] (权限: ADMFLAG_ROOT)

## 配置指令 (ConVar)

- `notify_map_next` 默认值: `1` - 终局开始后提示投票下一张地图的方式. \n0=不提示, 1=聊天栏, 2=屏幕中央, 4=弹出菜单.
