## 配置指令 (ConVar)

- `sm_tankheadshot_enable` 默认值: `1` - 插件开关. 0=禁用, 1=启用
- `sm_tankheadshot_prompt` 默认值: `3`
- `sm_tankheadshot_numberMultiple` 默认值: `1` - 是否启用伤害倍数随人数变化,禁用将一直使用默认倍数. 0=禁用, 1=启用
- `sm_tankheadshot_damagemultiple` 默认值: `10`
- `sm_tankheadshot_damagemultiple4` 默认值: `20`
- `sm_tankheadshot_damagemultiple8` 默认值: `40`
