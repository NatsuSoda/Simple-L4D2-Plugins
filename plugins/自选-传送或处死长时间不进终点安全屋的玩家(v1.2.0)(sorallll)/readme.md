# 自选-传送或处死长时间不进终点安全屋的玩家(v1.2.0)(sorallll)

## 简介

1:野生作者小改,闲置的玩家也能看到传送或处死倒计时等提示和听到倒计时声音了.

## 可用指令

- `sm_warpend` [管理员指令] (权限: ADMFLAG_RCON)
- `sm_st` [管理员指令] (权限: ADMFLAG_ROOT)

## 配置指令 (ConVar)

- `st_allow` 默认值: `1` - 启用传送幸存者插件? 0=禁用, 1=启用.
- `st_modes` 默认值: ``
- `st_modes_off` 默认值: ``
- `st_modes_tog` 默认值: `0`
- `st_enable` 默认值: `3` - 在什么情况下启用传送? 1=终点安全屋, 2=终点救援车, 3=两者.
- `st_type` 默认值: `1` - 倒计时结束后没有进入安全区的幸存者如何处理. 1=传送, 2=处死.
- `st_time` 默认值: `15` - 设置倒计时的时间/秒.
- `st_min_percent` 默认值: `50`
