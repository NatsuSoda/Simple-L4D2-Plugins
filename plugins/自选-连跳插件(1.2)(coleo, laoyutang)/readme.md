# 自选-连跳插件(1.2)(coleo, laoyutang)

## 简介

A fully featured bunnyhop plugin for Left

## 可用指令

- `sm_bhop`
- `sm_onrb`
- `sm_offrb`
- `sm_rb` [管理员指令] (权限: ADMFLAG_ROOT)

## 配置指令 (ConVar)

- `l4d_bunnyhop_mode` 默认值: `1`
- `l4d_bunnyhop_enable` 默认值: `0`
- `l4d_bunnyhop_multiplier` 默认值: `50` - 每次连跳成功后奖励玩家加速的值.
- `l4d_bunnyhop_limit` 默认值: `500` - 连跳速度上限
