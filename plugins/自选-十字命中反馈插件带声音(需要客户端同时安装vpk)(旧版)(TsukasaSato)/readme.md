# 自选-十字命中反馈插件带声音(需要客户端同时安装vpk)(旧版)(TsukasaSato)

## 简介

ConVar plugin_enable;

## 配置指令 (ConVar)

- `sm_hitsound_showtime` 默认值: `0.3`
- `sm_hitsound_mp3_headshot` 默认值: `hitsound/headshot.mp3` - 爆头音效的地址
- `sm_hitsound_mp3_hit` 默认值: `hitsound/hit.mp3` - 击中音效的地址
- `sm_hitsound_mp3_kill` 默认值: `hitsound/kill.mp3` - 击杀音效的地址
- `sm_hitsound_pic_headshot` 默认值: `overlays/head` - 爆头图标的地址
- `sm_hitsound_pic_hit` 默认值: `overlays/body` - 击中图标的地址
- `sm_hitsound_pic_kill` 默认值: `overlays/head` - 击杀图标的地址
- `sm_hitsound_sound_enable` 默认值: `1`
- `sm_hitsound_pic_enable` 默认值: `1`
- `sm_blast_damage_enable` 默认值: `0`
- `sm_hitsound_enable` 默认值: `1`
