# 自选-禁用游戏自带的投票(v1.0.0)(豆瓣酱な)

## 简介

阻止游戏自带的投票功能.

## 配置指令 (ConVar)

- `l4d2_enabled_change_alltalk` 默认值: `0` - 启用全局通话投票? 0=禁用, 1=启用.
- `l4d2_enabled_change_chapter` 默认值: `0` - 启用更换章节投票? 0=禁用, 1=启用.
- `l4d2_enabled_change_difficulty` 默认值: `0` - 启用更换难度投票? 0=禁用, 1=启用.
- `l4d2_enabled_change_mission` 默认值: `0` - 启用开始新图投票? 0=禁用, 1=启用.
- `l4d2_enabled_change_playerkick` 默认值: `0` - 启用踢出玩家投票? 0=禁用, 1=启用.
- `l4d2_enabled_change_restartgame` 默认值: `0` - 启用重新开始投票? 0=禁用, 1=启用.
- `l4d2_enabled_change_returntolobby` 默认值: `0` - 启用返回大厅投票? 0=禁用, 1=启用.
