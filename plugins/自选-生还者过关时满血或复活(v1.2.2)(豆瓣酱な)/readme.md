# 自选-生还者过关时满血或复活(v1.2.2)(豆瓣酱な)

## 简介

生还者过关时自动满血或复活.

## 可用指令

- `sm_hp`

## 配置指令 (ConVar)

- `l4d2_survivor_transition_Enabled` 默认值: `1` - 启用生还者过关时自动满血或复活? 0=禁用, 1=启用.
- `l4d2_survivor_transition_defaults` 默认值: `1` - 设置开启或关闭生还者过关满血或复活. 0=关闭, 1=开启.
- `l4d2_survivor_transition_IsStand` 默认值: `100`
- `l4d2_survivor_transition_IsDeath` 默认值: `80`
- `l4d2_survivor_transition_Hanging` 默认值: `80`
- `l4d2_survivor_transition_Ground` 默认值: `80`
