# 自选-坦克或女巫出现时根据生还者人数增加血量(v1.7.8)(Visor,豆瓣酱な)

## 简介

Announce in chat and via a sound when a Tank has spawned

## 配置指令 (ConVar)

- `l4d2_tank_Switch` 默认值: `1` - 启用坦克出现时根据生还者人数而增加血量? 0=禁用, 1=启用.
- `l4d2_tank_prompt` 默认值: `3`
- `l4d2_tank_Multiples` 默认值: `0.8;1.0;1.5;2.0`
- `l4d2_tank_minimum` 默认值: `4000`
- `l4d2_tank_health` 默认值: `2500` - 设置每多一个生还者坦克所增加的血量.
- `l4d2_witch_Switch` 默认值: `1` - 启用女巫出现时根据生还者人数而增加血量? 0=禁用, 1=启用.
- `l4d2_witch_Multiples` 默认值: `0.8;1.0;1.5;2.0`
- `l4d2_witch_minimum` 默认值: `1000`
- `l4d2_witch_health` 默认值: `200` - 设置每多一个生还者女巫所增加的血量.
