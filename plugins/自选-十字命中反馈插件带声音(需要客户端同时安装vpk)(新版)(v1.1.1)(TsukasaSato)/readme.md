# 自选-十字命中反馈插件带声音(需要客户端同时安装vpk)(新版)(v1.1.1)(TsukasaSato)

## 简介

自定义击中和击杀的图标、声音、时长

## 可用指令

- `sm_hitmarker`
- `sm_hitmarkers`
- `sm_hm`
- `sm_bhm`
- `sm_hmarker`
- `sm_crosshair_green`
- `sm_crosshair_green_change`
- `sm_crosshair_red`
- `sm_crosshair_red_change`
- `sm_crosshair_blue`
- `sm_crosshair_blue_change`

## 配置指令 (ConVar)

- `sm_hitsound_showtime_kill` 默认值: `2.0`
- `sm_hitsound_showtime_hit` 默认值: `0.2`
- `sm_hitsound_mp3_headshot` 默认值: `hitsound/headshot.mp3` - 爆头音效的地址
- `sm_hitsound_mp3_hit` 默认值: `hitsound/hit.mp3` - 击中音效的地址
- `sm_hitsound_mp3_kill` 默认值: `hitsound/kill.mp3` - 击杀音效的地址
- `sm_hitsound_classic_headshot` 默认值: `level/timer_bell.wav` - 无需下载内容的爆头音效的地址
- `sm_hitsound_classic_hit` 默认值: `buttons/button22.wav` - 无需下载内容的击中音效的地址
- `sm_hitsound_classic_kill` 默认值: `level/pointscored.wav` - 无需下载内容的击杀音效的地址
- `sm_hitsound_pic_headshot` 默认值: `overlays/head2` - 爆头图标的地址
- `sm_hitsound_pic_hit` 默认值: `overlays/body2` - 击中图标的地址
- `sm_hitsound_pic_kill` 默认值: `overlays/head2` - 击杀图标的地址
- `sm_hitsound_pic_hit_auto` 默认值: `overlays/body2` - 自动武器击杀图标的地址
- `sm_hitsound_sound_enable` 默认值: `1`
- `sm_hitsound_pic_enable` 默认值: `1`
- `sm_hitsound_pic_infect_kill_enable` 默认值: `1`
- `sm_hitsound_pic_infect_hit_enable` 默认值: `1`
- `sm_blast_damage_enable` 默认值: `0`
- `sm_fire_damage_enable` 默认值: `0` - 是否开启火烧反馈提示
- `sm_hit_infected_enable` 默认值: `1`
- `sm_kill_infected_enable` 默认值: `1`
- `sm_hitsound_melee_enable` 默认值: `1`
- `sm_hitsound_death_report_enable` 默认值: `1`
- `sm_hitsound_death_report_line` 默认值: `5`
- `sm_hitsound_death_report_time` 默认值: `5`
- `sm_hitsound_death_report_x` 默认值: `-0.315` - 单人模式仿COD的击杀播报的X坐标
- `sm_hitsound_death_report_y` 默认值: `0.4` - 单人模式仿COD的击杀播报的Y坐标
- `sm_hitsound_death_report_width` 默认值: `1.0` - 单人模式仿COD的击杀播报的宽度
- `sm_hitsound_death_report_height` 默认值: `0.22` - 单人模式仿COD的击杀播报的高度
- `sm_hitsound_enable` 默认值: `1`
