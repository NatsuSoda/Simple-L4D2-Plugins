# 自选-坦克特感女巫血量显示和坦克死亡提示(v1.0.4)(NiCo-op, Edited By Ernecio (Satanael))

## 简介

L4D Infected HP

## 配置指令 (ConVar)

- `l4d2_infectedhp` 默认值: `1` - 启用插件? 0=禁用, 1=启用.
- `l4d2_infectedhp_a` 默认值: `1` - 启用坦克卡死提示? 0=禁用, 1=聊天窗, 2=屏幕中下+聊天窗, 3=屏幕中下.
- `l4d2_infectedhp_bar` 默认值: `70`
- `l4d2_infectedhp_health` 默认值: `|` - 设置血量符号.
- `l4d2_infectedhp_damage` 默认值: ` ` - 设置去血符号.
- `l4d2_infectedhp_type` 默认值: `0` - 设置血条的显示位置和类型. 0=屏幕中心 1=屏幕中下.
- `l4d2_infectedhp_num` 默认值: `1` - 启用血量数字显示? 0=禁用, 1=启用.
- `l4d2_infectedhp_tank` 默认值: `1` - 启用坦克血条显示? 0=禁用, 1=启用.
- `l4d2_infectedhp_witch` 默认值: `1` - 启用女巫血条显示? 0=禁用, 1=启用.
- `l4d2_infectedhp_zmykey_boomer` 默认值: `1` - 启用 boomer 血条显示? 0=禁用, 1=启用.
