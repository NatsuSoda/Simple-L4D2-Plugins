# 自选-击杀特感和女巫奖励血量(v1.1.5)(豆瓣酱な)

## 简介

击杀特感和女巫奖励血量.

## 配置指令 (ConVar)

- `l4d2_health_oneshot_witch` 默认值: `20`
- `l4d2_survivor_health_Limit` 默认值: `100` - 设置幸存者获得血量奖励的最高上限.
- `l4d2_health_reviveSuccess` 默认值: `2`
- `l4d2_health_survivorRescued` 默认值: `3`
- `l4d2_health_healSuccess` 默认值: `15`
- `l4d2_health_defibrillator` 默认值: `20`
