# 自选-换图类(自动换图)(v1.0.5)(Alex Dragokas, fdxx, sorallll)

## 简介

自选-换图类(自动换图)(v1.0.5)(Alex Dragokas, fdxx, sorallll)

## 可用指令

- `sm_setnext` [管理员指令] (权限: ADMFLAG_RCON)

## 配置指令 (ConVar)

- `mapchanger_finale_change_type` 默认值: `12`
- `mapchanger_finale_failure_count` 默认值: `2` - 终局团灭几次自动换到下一张图
- `mapchanger_finale_random_nextmap` 默认值: `0` - 终局是否启用随机下一关地图

## 备注

- 配置文件: cfg/sourcemod/data/map_changer.cfg
