# 自选-排行榜插件(v2.21.30)(豆瓣酱な)(HUD的include提供者为 sorallll)

## 简介

HUD显示各种信息.

## 可用指令

- `sm_hud` \x04[提示]\x05你无权使用该指令.

## 配置指令 (ConVar)

- `l4d2_emshud_show_players_number` 默认值: `127`
- `l4d2_emshud_show_server_name_type` 默认值: `2` - 设置服务器名称的显示方式. -1=居中显示, 0=禁用, 1=从右到左显示, >1=连续显示多少次.
- `l4d2_emshud_show_server_name_time` 默认值: `10`
- `l4d2_emshud_show_server_name_number` 默认值: `63` - 显示服务器人数,路程,坦克和女巫刷新百分比. 0=禁用, 1=实体总数量, 2=服务器人数, 4=地图章节数量, 8=路程显示, 16=刷坦克路程, 32=刷女巫路程, 63=全部.
- `l4d2_emshud_show_server_time` 默认值: `7`
- `l4d2_emshud_ranking_Fake` 默认值: `0` - 排行榜显示电脑幸存者. 0=显示, 1=忽略.
- `l4d2_emshud_ranking_Info` 默认值: `15`
- `l4d2_emshud_ranking_Type` 默认值: `511`
- `l4d2_emshud_ranking_disp` 默认值: `63`
- `l4d2_emshud_ranking_rules` 默认值: `2` - 设置用什么类型排序. 1=血量, 2=特感, 3=爆头, 4=丧尸, 5=被黑, 6=友伤.
- `l4d2_emshud_kill_statistics` 默认值: `15`
- `l4d2_emshud_kill_reset_total` 默认值: `3`
