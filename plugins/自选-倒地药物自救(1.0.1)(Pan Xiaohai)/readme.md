## 配置指令 (ConVar)

- `l4d_selfhelp_incap` 默认值: `3` -  倒地自救 , 0:禁止, 1:止痛药、肾上腺素, 2:医疗包, 3:①②选项都可以
- `l4d_selfhelp_grab` 默认值: `3` -  被烟鬼控制自救, 0:禁止, 1:止痛药、肾上腺素, 2:医疗包, 3:①②选项都可以
- `l4d_selfhelp_pounce` 默认值: `3` -  被猎人控制自救, 0:禁止, 1:止痛药、肾上腺素, 2:医疗包, 3:①②选项都可以
