# 自选-生还者机关集合(v1.6)(alasfourom)

## 简介

Teleport Survivors To The Elevator After Time Passes

## 配置指令 (ConVar)

- `l4d_elevator_teleport_enable` 默认值: `1.0` - 启用电梯传送插件. 0 = 禁用, 1 = 启用.
- `l4d_elevator_teleport_time` 默认值: `15.0` - 设置电梯传送倒计时时间,时间归零时将传送幸存者.
- `l4d_elevator_teleport_maps` 默认值: `all`
- `l4d_elevator_teleport_entire` 默认值: `1` - 插件将传送所有幸存者,包括挂边和被控制的幸存者.
