# 自选-阻止游戏自带的部分提示(v1.1.2)(豆瓣酱な)

## 简介

屏蔽游戏自带的部分提示.

## 配置指令 (ConVar)

- `l4d2_text_msg` 默认值: `1` - 屏蔽游戏自带的闲置提示? 0=显示, 1=屏蔽.
- `l4d2_PZDmg_msg` 默认值: `1` - 屏蔽游戏自带的其它提示? 0=显示, 1=屏蔽.
- `l4d2_server_cvar` 默认值: `1` - 屏蔽游戏自带的ConVar更改提示? 0=显示, 1=屏蔽.
- `l4d2_player_death` 默认值: `1` - 屏蔽游戏自带的玩家死亡提示? 0=显示, 1=屏蔽.
- `l4d2_player_incapacitated` 默认值: `1` - 屏蔽游戏自带的玩家倒下提示? 0=显示, 1=屏蔽.
- `l4d2_player_disconnect` 默认值: `1` - 屏蔽游戏自带的玩家离开提示? 0=显示, 1=屏蔽.
- `l4d2_defibrillator_used` 默认值: `1` - 屏蔽游戏自带的使用电击器提示? 0=显示, 1=屏蔽.
