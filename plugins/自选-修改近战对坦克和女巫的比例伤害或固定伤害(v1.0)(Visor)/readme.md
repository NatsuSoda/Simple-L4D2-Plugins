# 自选-修改近战对坦克和女巫的比例伤害或固定伤害(v1.0)(Visor)

## 简介

Nerfes melee damage against tanks by a set amount of %

## 配置指令 (ConVar)

- `l4d2_melee_tank_damage` 默认值: `200.0`
- `l4d2_melee_tank_display` 默认值: `1` - 显示近战对坦克的伤害. 0=禁用, 1=显示.
- `l4d2_melee_witch_damage` 默认值: `-1.0`
- `l4d2_melee_witch_display` 默认值: `1` - 显示近战对女巫的伤害. 0=禁用, 1=显示.
