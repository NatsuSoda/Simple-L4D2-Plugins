# 自选-玩家指令!vt投票插件(v2.4.6)(豆瓣酱な)

## 简介

投票更改难度,踢出玩家,重启当前章节.

## 可用指令

- `sm_vt`

## 配置指令 (ConVar)

- `l4d2_vote_BanTime` 默认值: `10080` - 设置被投票踢出玩家的封禁时间/分钟. -1=仅踢出, 0=永久封禁.
- `l4d2_vote_Percent` 默认值: `60`
- `l4d2_vote_menuTime` 默认值: `25` - 设置同意或反对菜单的显示时间/秒.
- `l4d2_vote_difficulty` 默认值: `15`
- `l4d2_vote_target_player` 默认值: `0`
