# 自选-设置倒地次数为0时阻止不正常的心跳声(Sir, 豆瓣酱な)(内有作者原版)

## 简介

Allows control over certain sounds

## 配置指令 (ConVar)

- `l4d2_sound_heartbeatloop_flags` 默认值: `1` - 阻止幸存者不正常的心跳声音. 0=禁用, 1=启用.
- `l4d2_sound_vehicle_impact_heavy_flags` 默认值: `1` - 阻止幸存者死亡时的重击声音. 0=禁用, 1=启用.
- `l4d2_sound_incapacitatedinjury_flags` 默认值: `1` - 阻止幸存者死亡后播放的音乐. 0=禁用, 1=启用.
