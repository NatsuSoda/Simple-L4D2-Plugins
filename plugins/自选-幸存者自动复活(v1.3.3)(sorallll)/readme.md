# 自选-幸存者自动复活(v1.3.3)(sorallll)

## 简介

Survivor Auto Respawn

## 配置指令 (ConVar)

- `sar_respawn_survivor_time` 默认值: `10`
- `sar_respawn_survivor_time_add` 默认值: `2`
- `sar_respawn_survivor_time_max` 默认值: `60`
- `sar_respawn_survivor_limit` 默认值: `10` - 玩家每回合自动复活次数.
- `sar_respawn_survivor_bot` 默认值: `1` - 启用电脑幸存者自动复活? 0=禁用, 1=启用.
- `sar_respawn_survivor_idle` 默认值: `1` - 启用闲置幸存者自动复活? 0=禁用, 1=启用.
- `sar_respawn_survivor_daze` 默认值: `5` - 复活后多久无操作自动闲置. 0=禁用.
- `sar_respawn_slot0` 默认值: `0` - 主武器给什么 \n0=不给,131071=所有,7=微冲,1560=霰弹,30720=狙击,31=Tier1,32736=Tier2,98304=Tier0.
- `sar_respawn_slot1` 默认值: `0` - 副武器给什么 \n0=不给,131071=所有.如果选中了近战且该近战在当前地图上未解锁,则会随机给一把.
- `sar_respawn_slot2` 默认值: `0` - 投掷物给什么 \n0=不给,7=所有.
- `sar_respawn_slot3` 默认值: `0` - 槽位3给什么 \n0=不给,15=所有.
- `sar_respawn_slot4` 默认值: `0` - 槽位4给什么 \n0=不给,3=所有.
- `sar_give_type` 默认值: `0`
- `sar_remove_drop` 默认值: `1`
