# 可选-修复类(v1.5)修复高tick的服务器口水伤害异常高的问题(Visor, Sir, A1m`)

## 简介

Make the spit deal a set amount of DPS under all circumstances

## 配置指令 (ConVar)

- `l4d2_spit_dmg` 默认值: `-1.0` - Damage per tick the spit inflicts. -1 to skip damage adjustments
- `l4d2_spit_alternate_dmg` 默认值: `-1.0` - Damage per alternate tick. -1 to disable
- `l4d2_spit_max_ticks` 默认值: `28` - Maximum number of acid damage ticks
- `l4d2_spit_godframe_ticks` 默认值: `4` - Number of initial godframed acid ticks
