# 自选-战役使用对抗刷坦克和女巫配置(v2.4.1)(CanadaRox, Sir, devilesk, Derpduck, Forgetest)

## 简介

Sets a tank spawn and has the option to remove the witch spawn point on every map

## 可用指令

- `sm_tank_witch_debug_test`
- `sm_tank_witch_debug_profiler`
- `sm_tank_witch_debug_info` [管理员指令] (权限: ADMFLAG_KICK)

## 配置指令 (ConVar)

- `sm_tank_witch_debug` 默认值: `0` - Tank and Witch ifier debug mode
- `sm_tank_can_spawn` 默认值: `1` - Tank and Witch ifier enables tanks to spawn
- `sm_witch_can_spawn` 默认值: `1` - Tank and Witch ifier enables witches to spawn
- `sm_witch_avoid_tank_spawn` 默认值: `20` - Minimum flow amount witches should avoid tank spawns by, by half the value given on either side of the tank spawn
