# 自选-显示幸存者开关安全门或次数限制(v1.0.1)(豆瓣酱な)

## 简介

防止玩家玩安全门

## 配置指令 (ConVar)

- `l4d2_enabled_safeRoomDoor_a` 默认值: `1` - 启用幸存者开关安全门提示+开关安全门次数限制? 0=禁用, 1=启用, 2=禁用限制,只显示幸存者开关安全门.
- `l4d2_enabled_safeRoomDoor_b` 默认值: `1`
- `l4d2_enabled_safeRoomDoor_c` 默认值: `5`
- `l4d2_enabled_safeRoomDoor_d` 默认值: `1`
- `l4d2_enabled_safeRoomDoor_e` 默认值: `5` - 设置每个章节幸存者开关安全门的次数.
- `l4d2_enabled_safeRoomDoor_f` 默认值: `2.0`
