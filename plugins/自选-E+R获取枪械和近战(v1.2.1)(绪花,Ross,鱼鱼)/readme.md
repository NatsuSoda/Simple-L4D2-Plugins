## 配置指令 (ConVar)

- `l4d2_meleedefault` 默认值: `1` - 默认是否开启副武器菜单
- `l4d2_initialgundefault` 默认值: `1` - 默认是否开启小枪菜单
- `l4d2_safearea` 默认值: `1` - 默认是否限制安全区域内使用
- `l4d2_AdvancedGunOpen` 默认值: `0` - 默认是否开启大枪菜单
